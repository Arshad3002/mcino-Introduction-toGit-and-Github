# Simple Interest Calculator

A simple shell script to calculate simple interest.

## Description

This project contains a Bash script that calculates simple interest given the principal amount, annual rate of interest, and time period in years.

## Features

- Calculate simple interest
- User-friendly input
- Displays the calculated interest

## Formula

Simple Interest = (Principal × Rate × Time) / 100
